# audacity-keys

Different keymaps for [Audacity](https://www.audacityteam.org/).

## Keymaps

- [default](./default-keys.xml) keys
- [piecioshka](./piecioshka-keys.xml) keys
- [extratone](https://gist.github.com/extratone/eb2ae65c8827a0a9e3f7792f0e3a021e) keys `(Gist)`

See a difference between keymaps by comparing:

```bash
diff -u default-keys.xml piecioshka-keys.xml
```
